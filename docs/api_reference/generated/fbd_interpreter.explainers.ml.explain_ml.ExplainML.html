fbd_interpreter.explainers.ml.explain_ml.ExplainML¶

class fbd_interpreter.explainers.ml.explain_ml.ExplainML(model: Any, task_name: str, tree_based_model: bool, features_name: List[str], features_to_interpret: List[str], target_col: str, out_path: str)[source]¶

Class that contains different interpretability techniques to explain the model training (global interpretation) and its predictions (local interpretation).

model¶

Model to compute predictions using provided data, Any scikit-learn model, model.predict(data) must work Note that you can also use a pyspark model if you are only using SHAP interpretation

Type

scikit-learn model or Pyspark model

task_name¶

Task name: choose from supported_tasks in config/config_{type_env}.cfg

Type

str

tree_based_model¶

If True, we use Tree SHAP algorithms to explain the output of ensemble tree models

Type

bool

features_name¶

List of features names used to train the model

Type

List[str]

features_to_interpret¶

List of features to interpret using pdp, ice and ale

Type

List[str]

target_col¶

name of target column

Type

str

out_path¶
